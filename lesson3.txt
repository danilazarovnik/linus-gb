2. Управление группами:
a) создать группу с использованием утилит и в ручном режиме;

parallels@ubuntu-gb:~$ sudo groupadd team1
parallels@ubuntu-gb:~$ sudo addgroup team2
Добавляется группа «team2» (GID 1003) ...
Готово.


b) попрактиковаться в смене групп у пользователей;

parallels@ubuntu-gb:~$ sudo usermod -G team1 user3
parallels@ubuntu-gb:~$ sudo usermod -G team2 user3


c) добавить пользователя в группу, не меняя основной;

parallels@ubuntu-gb:~$ sudo usermod -aG sudo user1


d) удалить пользователя из группы.

parallels@ubuntu-gb:~$ sudo deluser user3 team2
Удаляется пользователь «user3» из группы «team2» ...
Готово.


3. Создать пользователя с правами суперпользователя. Сделать так, чтобы sudo не требовал пароль для выполнения команд.

parallels@ubuntu-gb:~$ sudo usermod -aG sudo user1
parallels@ubuntu-gb:~$ sudoedit /etc/sudoers
%sudo ALL=(ALL) NOPASSWD:ALL
